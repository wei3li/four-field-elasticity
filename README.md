# Code and results for the paper *A stable four-field mixed finite element method*

This repository contains scripts and data for the numerical experiments in the paper:

**A stable four-field mixed finite element method for incompressible nonlinear elasticity**

## Contents
- `scripts/`: scripts for running simulations
- `results/`: (lightweight) output data from simulations
- `meshes/`: geometry files to create meshes in `Gmsh`
